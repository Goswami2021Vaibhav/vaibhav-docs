# Interview Prep

Question banks for technical interviews — JavaScript, TypeScript, React, Next.js, and more, growing as I study.

This isn't a tutorial series. It's a **self-test vault**: each topic is trimmed down to the questions that actually get asked, one collapsible card per question so I can read it, try to answer it out loud, then reveal the answer to check myself.

## What's Here

- **[JavaScript](/interview/javascript/javascript-fundamentals)** — fundamentals, execution context, scope, closures, `this`, async, the event loop, and more, split topic by topic

More subjects (TypeScript, React, Next.js) will be added as notes get written.

## Why This Exists

Knowing an answer and being able to say it clearly under pressure are two different skills. Every question here is picked because it's genuinely likely to come up — not padded out for volume — so a full pass through a topic stays realistic to actually do before an interview, not just something to skim once and forget.
