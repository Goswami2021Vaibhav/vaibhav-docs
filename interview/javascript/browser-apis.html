Browser APIs

1. What is the DOM, how does the browser construct it, and what's the difference between getElementById(), querySelector(), and querySelectorAll()?​

2. What is the difference between innerHTML, innerText, and textContent, and how do you create/append/remove DOM elements?​

3. What is DOM Traversal (parentNode/parentElement, children/childNodes), and what is a Document Fragment used for?​

4. What causes DOM reflow/repaint, and what is the Virtual DOM (and how does it compare to the Real DOM)?​

5. What is the Browser Object Model (BOM), and how does it differ from the DOM (window, location, history, navigator)?​

6. What are Event Bubbling, Event Capturing, and Event Propagation, and what do stopPropagation()/preventDefault() do?​

7. What is Event Delegation, why is it useful, and how does it improve performance for dynamically added elements?​

8. What is the difference between localStorage, sessionStorage, and Cookies, and when should you use each?​

9. Why shouldn't sensitive information be stored in localStorage, and what are HttpOnly Cookies?​

10. What is the Fetch API, why is it preferred over XMLHttpRequest, and why doesn't fetch() reject on HTTP 404/500?​

11. What is AbortController, and how would you cancel a Fetch request (e.g., on React component unmount)?​

12. How would you implement Event Delegation for a large table, or prevent multiple form submissions?​

13. How would you securely store authentication information in the browser?​
